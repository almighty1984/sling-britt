Menu:
    
    F1                 - Edit
    F2                 - Game

Edit:

    Move mouse to top of window to access menu
    Press del on level in list to delete

    WASD               - move camera / selected
    Space              - show tile set
    Scroll / 1-5       - brush size
    Ctrl               - select    
    X                  - move selected
    C                  - copy selected
    Ctrl-Z             - undo
    Ctrl-Shift-Z       - redo
    Alt                - hide menu
    Ctrl-S             - save level
    Ctrl-Shift-S       - save level as    

Game:

    Arrows             - move
    Z                  - sprint
    X                  - jump
    C                  - melee
    Ctrl               - move camera
