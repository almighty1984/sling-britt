Edit:

    Shift              - show menu
    W, A, S, D         - move camera / selected
    Space              - show tile set
    Scroll / 1 - 5     - brush size
    Ctrl               - select    
    X                  - move selected
    C                  - copy selected
    Ctrl-Z             - undo
    Ctrl-Shift-Z       - redo
    Alt                - hide menu
    Ctrl-S             - save level
    Ctrl-Shift-S       - save level as    

Game:

    Arrows             - move
    Z                  - sprint
    X                  - jump
    C                  - melee
    Ctrl               - move camera
